# 可爱风格签名生成器

![cover](./assets/cover.png)

---

## 📖 简介

**Cute Signature Generator** 是一个基于 OOMOL 平台的签名生成工具，能够创建带有可爱卡通装饰的手写风格签名。支持小猫、兔子、星星、爱心等多种萌趣元素，打造专属的个性化签名。

在 OOMOL Hub 上使用: <https://hub.oomol.com/package/cute-signature-generator>

---

## ✨ 功能特性

- 🎨 **可爱风格设计** - 卡通装饰的萌趣风格
- ✍️ **手写风格** - 细笔触手写效果，笔画优雅
- 🐱 **丰富装饰元素** - 可爱动物（小猫、兔子、小鸟、小狗、小熊）、星星、爱心、花朵、闪光等
- 🖼️ **多种宽高比** - 支持多种宽高比（21:9, 16:9, 3:2, 4:3, 5:4, 1:1, 4:5, 3:4, 2:3, 9:16）
- 📐 **多分辨率输出** - 多种分辨率选项（1K, 2K, 4K）
- 🎯 **个性化定制** - 可自定义名字和装饰元素

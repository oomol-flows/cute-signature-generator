## 📖 Introduction

**Cute Signature Generator** is a signature generation tool based on the OOMOL platform that creates handwritten-style signatures with adorable cartoon decorations. It supports various cute elements like cats, bunnies, stars, hearts, and more to create personalized kawaii-style signatures.

---

## ✨ Features

- 🎨 **Kawaii aesthetic** - Cute style design with cartoon decorations
- ✍️ **Handwriting style** - Fine-tip pen handwriting effect with elegant strokes
- 🐱 **Rich decoration elements** - Cute animals (cats, bunnies, birds, puppies, bears), stars, hearts, flowers, sparkles
- 🖼️ **Multiple aspect ratios** - Support for multiple aspect ratios (21:9, 16:9, 3:2, 4:3, 5:4, 1:1, 4:5, 3:4, 2:3, 9:16)
- 📐 **Multiple resolution output** - Multiple resolution options (1K, 2K, 4K)
- 🎯 **Personalized customization** - Customizable name and decoration elements
